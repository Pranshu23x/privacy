Privacy Policy

Effective Date:11/07/2025

This Chrome extension, Time Complexity Analyzer, does not collect, store, or share any user data.

Data Usage

All code entered by the user is processed locally and securely sent to the official Google Gemini API over HTTPS for the sole purpose of time complexity analysis.

No personally identifiable information (PII), browsing history, or other private data is accessed or retained by this extension.

Third-Party Services

This extension uses the Gemini API from Google to analyze code. Please refer to Google's Privacy Policy for details on how they handle data.
